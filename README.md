# House-Price-Predictor
An intelligent web application that predicts property prices Machine Learning. Built with Flask and Random Forest algorithm, trained on 13,000+ real property records across 376 locations.
An AI-powered web application that predicts house prices in Bengaluru using Machine Learning.

## ✨ Features

- 🎯 **94.3% Model Accuracy** - Random Forest ML model
- 📍 **376+ Locations** - Covers all major Bengaluru areas
- 📊 **13,000+ Properties** - Trained on real data
- 🎨 **Modern UI** - Beautiful gradient design
- ⚡ **Instant Predictions** - Real-time price estimates


## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **ML Model:** Random Forest Regressor
- **Frontend:** HTML5, CSS3, JavaScript
- **Data Processing:** Pandas, NumPy, Scikit-learn

## 📋 Prerequisites

- Python 3.8+
- pip package manager
